General description of the task

In this project we will use AJAX to fetch the data from the web. This time the data is not a static file, but rather a stream of data returned by a live server.
From a programmers point of view it makes no difference which data source is used, the process is the same:

1. First we ask for the data (Ajax call)
2. Save the results into a variable and select what data we want to use
3. Display the selected data and make it look nice

Project will be published in GitHub and deployed to Netlify.
Comment every line of JavaScript code you write.
DO not use external libraries.
Use CSS-styles to style your page.
All event handlers should be added dynamically! no hard coded onclick -listeners!

Features:
  • Generates query from set station and train count
  • If dropdown -menu is used, search box empties automatically.
  • Checkboxes have 1sec delay before submitting.
  • If sorting is requested by non-selected times, requested times are displayed.
  • Sort menu also updates used sorting when checkboxes are clicked.
  • Translates station codes into station names.
  • Station search by station name.
    • Refined, now gives also suggestions and is automatically fixes first letter, if lowercase.
Issues:
  • When sorting trains by departure time for example in Tampere while both, arrivals and departures are selected, sorting might not work properly when starting/terminating trains start to appear on array.
